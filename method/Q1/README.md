### Q1. Grade Result

Write a program which will ask the user to enter his/her marks (out of 100). Define a method that will display grades according to the marks entered as below:

```
Marks Grade
91-100 AA
81-90 AB
71-80 BB
61-70 BC
<=60 Fail
```

**Sample input:**

```
70
23
93
```

**Sample output:**

```
BC
Fail
AA
```
